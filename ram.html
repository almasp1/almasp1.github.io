Detta är ram sidan

RAM står för Random Access Memory. Ramminnet är datorns korttidsminne om inget ramminne är finns i datorn så kommer den inte att starta. Det är ett snabbt minne som måste ha ström för att fungera. När ett program startas laddas det in från hårddisken och sparas i ramminnet. Datorn måste ha tillräckligt mycket ramminne för att aktiva program och dokument ska få plats. Blir ramminnet fullt kommer Windows tillfälligt att flytta delar av innehållet till hårddisken. Eftersom en hårddisk är mycket långsammare än ett ramminne kommer detta att påverka prestandan. En dator som alltid ska vara snabb behöver mycket ramminne. Ett ramminne måste alltid vara strömsatt för att fungera. Det innebär att det töms så fort datorn stängs av. Hur snabbt ett ramminne är påverkar hur snabbt datorn kan prestera, därför är det viktigt att köpa ramminne som passar i din dator.
